## Get started

1. Pull Repo

   ```bash
   git clone
   ```
2. Install dependencies

   ```bash
   npm install
   ```
3. Login Your Eas Account

   ```bash
   eas login
   ```

4. Build development

   ```bash
   eas build --profile development --platform android
   ```

5. Install the apk

6. Start the app

   ```bash
    npx expo start
   ```
